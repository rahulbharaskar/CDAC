Overview of SQL.

1) Does ANSI recognize the data type DATE? 
ANS:
ANSI SQL recognizes the DATE data type. It is used to store calendar dates (year, month, and day). Different databases that follow ANSI SQL may implement it slightly differently, but the standard DATE type is widely supported.
